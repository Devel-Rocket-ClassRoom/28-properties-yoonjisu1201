# BankAccount

은행 계좌를 표현하는 `BankAccount` 클래스를 작성하세요.

**요구사항**

1. `AccountNumber` 속성: 계좌번호 (읽기 전용, 생성 시에만 설정 가능)    // 생성자
2. `OwnerName` 속성: 예금주 이름 (읽기/쓰기 가능)
3. `Balance` 속성: 잔액 (읽기 전용, 클래스 내부에서만 변경 가능)
   - 잔액은 0 미만이 될 수 없음 (음수 설정 시 0으로 유지)
4. `Deposit(int amount)` 메서드: 입금 (양수만 허용)
5. `Withdraw(int amount)` 메서드: 출금 (잔액 초과 시 출금 불가, false 반환)

**테스트 시나리오**

- 계좌 생성: 계좌번호 "123-456", 예금주 "홍길동"
- 50000원 입금
- 20000원 출금
- 100000원 출금 시도 (잔액 부족으로 실패)
- 최종 잔액 확인

## 예상 실행 결과

```
계좌번호: 123-456
예금주: 홍길동
잔액: 0원
--- 50000원 입금 ---
잔액: 50000원
--- 20000원 출금 ---
출금 성공: True
잔액: 30000원
--- 100000원 출금 시도 ---
출금 성공: False
잔액: 30000원
```
